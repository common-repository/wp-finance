Manage your financial records from your Wordpress admin panel!

== Description ==

WP Finance provides easy to use interface to create and manage your financial records in your personal or commercial blog. All financial statements can be later disclosed to your blog visitors by embedding short code into the blog post or custom page. 

You can help us to improve WP Finance by suggesting new features, tips how we could increase your productivity while using WP Finance or report existing bugs [here](http://mindomobile.com/contact-us/). We appreciate any help!

Become our [**Facebook**](http://www.facebook.com/MindoMobileSolutions) fan to keep up with the latest updates. You can also find us on **Google+** circles [here](https://plus.google.com/116940685804905541644).

ENJOY!

NOTE: New releases of the plugin always have numerous of untranslated text strings that are waiting to be translated. Please, submit your translations if you wish to have them included into official release of plugin.

== Installation ==

**There are 2 ways to install WP Finance on your blog:**

Upload the plugin to your blog plugin directory using ftp client or web file manager and activate it or;

Visit your blog admin panel and:

1. Click Plugins in the menu
2. Click Add New
3. In search field type "wp finance" (without quotes)
4. Find WP Finance in the list and click Install Now.

== Upgrade Notice ==

To upgrade plugin, please copy and replace all existing files in wp-finance directory. All necessary adjustments by the plugin will be done automatically. You can also update plugin using standard WordPress Update Manager!

== Frequently Asked Questions ==

= I have some troubles with plugin what should I do? =

You can contact us by email wpfinance@mindomobile.com or you can leave us a message in contact form http://mindomobile.com/contact-us/. Moreover, you can also post a thread in [Wordpress support forum](http://wordpress.org/support/plugin/wp-finance) but you should keep in mind it might take some time to respond to your problem.

= I prepared the translations for the WP Finance. How I can upload it? =

You can simply send the translation files to our email wpfinance@mindomobile.com and we would pack it within the next release of the plugin. Please, do not forget to include *.po and *.mo files as well as your Wordpress username for credits!
